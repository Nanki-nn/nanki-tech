# Personal Site

A personal technical blog, GitHub project showcase, and portfolio site based on the structure of [PRISM](https://github.com/xyjoey/PRISM).

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Edit Content

Most site content lives in `content/`:

- `content/config.toml`: name, title, social links, navigation
- `content/bio.md`: homepage introduction
- `content/blog.toml`: technical writing entries
- `content/projects.toml`: GitHub projects
- `content/works.toml`: portfolio work

## Build

```bash
npm run build
```

The site is configured for static export.
